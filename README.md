# 自动线稿绘制器

这是一个基于Windows的桌面工具，支持将导入图片或中英文文字转换为线稿，在界面中实时预览，并通过鼠标自动完成绘制。
如果你是一个你画我猜，杀戮尖塔等游戏的爱好者，但是苦于自己无法绘制出简单美观的线稿来与朋友互动，那么这个小软件也许可以帮到你。

## 功能

- 基于canny+sobel算法，将导入的图片转换为可调整精度的线稿路径。
- 支持输入中英文文字并转换为字体轮廓线稿。
- 在线稿生成后于右侧实时预览。
- 框选屏幕目标区域，并在保持比例的前提下适配线稿。
- 支持选择使用鼠标左键或右键进行自动绘制。
- 支持通过界面按钮或 `Esc`热键停止绘制。

## 使用方式

- /dist文件夹中有Auto-drawpic.exe，下载后双击进入（部分机型可能需要管理员启动）
- 自由选择导入图片或输入中英文文字，输入后点击生成线稿，并可调整图片精度或字体类型、字号大小
- 选择使用左键或右键绘制（你画我猜可使用左键，杀戮尖塔2使用右键绘制）
- 选择绘制速度与开始绘制倒计时
- 点击开始绘制后框选绘画区域，绘制倒计时后在区域内绘制等比例缩放的线稿预览
- 增加了可调的给画速度值，0-100时保持给画速度与上一版同步以保证最圭的绘制效果:100-200线段插值变稀，牺牲部分平滑换取绘制速度:200-300区间减少起笔婷待、减少按下等待，进一步减少插值间距以换取最快绘图速度
- 新增应用图标ico为千早爱音
